# Quiz App Landing Page

Landing page for the quiz app, built with [Next.js](https://nextjs.org) and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

- `npm run dev` — start the development server
- `npm run build` — create a production build
- `npm start` — serve the production build

## Structure

- `src/app/(public)` — public pages (home, about, privacy, terms, success)
- `src/app/(auth)` — auth pages (sign in, sign up, password reset, email verification)
- `src/components` — UI components
- `src/assets/images` — image assets
